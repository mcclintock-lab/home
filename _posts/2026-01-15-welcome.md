We're excited to launch our new website! Built on GitHub Pages, this platform will serve as a hub for sharing updates about our ocean planning tools, announcing new features, and connecting with the marine spatial planning community.

Our new site features:

- Improved navigation and mobile responsiveness
- Detailed project information
- Team profiles
- This blog for sharing updates and insights

## Looking Forward

We'll be using this blog to share:

- Product updates and new features
- Case studies from SeaSketch, eCatch, and SeaMonitor users
- Technical insights and tutorials
- Ocean planning news and resources

Stay tuned for more updates!
